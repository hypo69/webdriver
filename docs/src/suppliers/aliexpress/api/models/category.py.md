# Модуль `category.py`

## Обзор

Модуль определяет структуру данных для категорий товаров, используемых в API AliExpress. Он содержит классы `Category` и `ChildCategory`, представляющие собой базовую категорию и дочернюю категорию соответственно.

## Подробней

Этот модуль предоставляет классы для представления категорий товаров на AliExpress. Он определяет атрибуты, общие для всех категорий (`category_id`, `category_name`), а также атрибут, специфичный для дочерних категорий (`parent_category_id`). Эти классы используются для структурирования данных, получаемых от API AliExpress, и облегчения работы с ними.

## Классы

### `Category`

**Описание**: Базовый класс для представления категории товара.

**Принцип работы**: Класс `Category` служит основой для представления любой категории товара на AliExpress. Он содержит идентификатор категории (`category_id`) и её название (`category_name`).

**Атрибуты**:

-   `category_id` (int): Уникальный идентификатор категории.
-   `category_name` (str): Название категории.

### `ChildCategory(Category)`

**Описание**: Класс для представления дочерней категории товара, наследуемый от класса `Category`.

**Принцип работы**: Класс `ChildCategory` расширяет класс `Category`, добавляя информацию о родительской категории. Это позволяет устанавливать иерархические связи между категориями товаров.

**Наследует**:

-   `Category`: Класс `ChildCategory` наследует атрибуты `category_id` и `category_name` от класса `Category`.

**Атрибуты**:

-   `parent_category_id` (int): Идентификатор родительской категории.
-   `category_id` (int): Уникальный идентификатор категории.
-   `category_name` (str): Название категории.

```python
class Category:
    category_id: int
    category_name: str


class ChildCategory(Category):
    parent_category_id: int
